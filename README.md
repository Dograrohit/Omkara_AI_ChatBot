🕉️ OMkara AI

Your Intelligent AI Assistant, Built with MERN + React + Gemini/OpenAI APIs

OMkara AI is a modern ChatGPT-inspired AI chatbot designed to deliver fast, intelligent, 
and beautiful conversations. It features a sleek interface, real-time AI responses, chat history,
authentication, image generation support, dark mode, and much more.

Whether you're asking questions, generating content, or brainstorming ideas,
OMkara AI is built to make AI feel natural and accessible.

✨ Features :- 

🤖 AI-powered conversations
💬 Persistent chat history
🔐 Secure user authentication
🌙 Dark & Light mode
🖼️ AI image generation support
📱 Responsive design
⚡ Fast React-based UI
📂 Multiple chat sessions
🔍 Search through previous chats

🚀 Installation :-

Clone the repository

git clone https://github.com/yourusername/omkara-ai.git

Navigate into the project

cd omkara-ai

Install dependencies

npm install

Create a .env file

MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=
OPENAI_API_KEY=
CLOUDINARY_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

Start the development server

npm run dev
